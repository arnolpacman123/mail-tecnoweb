CONSIDERACIONES

- No ingresar caracteres especiales, la letra Ñ (ñ), ni caracteres con tíldes.

- El formato para fechas es AAAA-MM-DD, por ejemplo 1998-03-14.

- El sexo es el caracter 'F' para femenino y 'M' para masculino.

- Solo se aceptaran numeros de celular de hasta 12 digitos.

- Solo se aceptan contraseñas de 8 a 10 caracteres, minimo una minuscula, mayuscula, numero y caracter especial.

Casos de uso Sistema para Conodominios Protecnosc

CU1: GESTIONAR PERFIL
Registrar Usuario
FORMATO => RESUSER:loginAdmin:PassAdmin:Nom:App:CI:Sexo:Telf:Email:ocupacion:login_user:Pass_user:Tipo{ADMINISTRADOR|PROPIETARIO}|[[cargo{DUENO|INQUILINO}]|fechafin]
EJEMPLO1 => RESUSER:bolita@condiminio.com:123:Simon:Sagredo Cordova:9958112:M:76212101:simon@gmail.com:Estudiante:simon@condominio.com:simon123:ADMINISTRADOR
EJEMPLO2 => RESUSER:bolita@condominio.com:123:Juan:Perez:1234:M:70001123:juan@gmail.com:profesor:juan@condiminio.com:juan123:PROPIETARIO|DUENO
EJEMPLO3 => RESUSER:bolita@condominio.com:123:ericka:Quiroz:4321:F:7654321:ericka@gmail.com:contadora:ericka@condominio.com:ericka123:PROPIETARIO|Inquilino|2022-12-01
Modificar contraseña de acceso
FORMATO => MODRESACC:login:New_pass:Old_pass
EJEMPLO => MODRESACC:juan@condominio.com:juancito123:juan123
Eliminar Acceso a un usuario //tambien los borra del id de casa y del propietario
FORMATO => DELRESUSER:login:Pass:Acc_Elim
EJEMPLO => DELRESUSER:soto@condominio.com:juan123:yordi@condominio.com
Modificar elementos de un usuario
FORMATO => MODUSER:login:pass:telf:email:ocupacion
EJEMPLO => MODUSER:juan@condominio.com:juan123:123456:juan@gmail.com:profesor particular
Listar usuario
FORMATO => LISUSER:login:pass:[ADMINISTRADOR|COMITE|propietario{dueño}|*]
EJEMPLO => LISUSER:juan@condominio.com:juan123:*
Registrar un usuario a una casa como residente //Para que se añaden los inquilinos primero se debe registrar el propietario dueño
FORMATO => ADDUSERCASA:login:Pass:nomCondominio:nroArea:nroCasa
EJEMPLO => ADDUSERCASA:ericka@condominio.com:ericka123:1
Modificar cargo en un usuario //si modificas el cargo es porque eres dueño propietario de una casa
FORMATO => MODCARGO:loginAdmin:PassAdmin:loginUser:cargo:[fecha]
EJEMPLO1 => MODCARGO:bolita@condominio.com:123:juan@condominio.com:secretario
EJEMPLO2 => MODCARGO:bolita@condominio.com:123:ericka@condominio.com:presidente:2022-12-01
CU2: GESTIONAR CONDOMINIO
Registrar un condominio
FORMATO => REGCON:loginAdmin:passAdmin:nom:ubic:datosUB:dimension
EJEMPLO => REGCON:bolita@condominio.com:123:Laguna Azul:La Guardia:Santa Cruz, Bolivia:100 hect
Modificar datos de un condominio
FORMATO => MODCON:loginAdmin:passAdmin:nomOriginal[|nuevoNom]:ubic:datosUB:dimension
EJEMPLO1 => MODCON:bolita@condominio.com:123:Laguna Azul|Condominio Laguna:km 14:Santa Cruz, Bolivia:500 hect
EJEMPLO2 => MODCON:bolita@condominio.com:123:Laguna Azul:La Guardia:Santa Cruz, La Guardia:100 hect
Ver informacion de un condominio especifico
FORMATO => VERCON:login:pass:nom
EJEMPLO => VERCON:juan@condominio.com:juan123:Laguna Azul
Listar todos los condominios que tiene registrado el sistema
FORMATO => LISCON:login:pass
EJEMPLO => LISCON:juan@condominio.com:juan123
Eliminar el registro de un condominio por su nombre
FORMATO => DELCON:loginAdmin:passAdmin:nom
EJEMPLO => DELCON:bolita@condominio.com:123:Laguna Azul
CU3: GESTIONAR CASAS Y AREAS COMUNES
Registrar un area comun
FORMATO => REGMAN:loginAdmin:passAdmin:nomCondominio:nroArea:tipo{urbano|area verde|parque|rural}:dimension
EJEMPLO => REGMAN:bolita@condominio.com:123:Laguna Azul:1:urbano:1000x1000
Modificar un area comun especifico
FORMATO => MODMAN:loginAdmin:passAdmin:nomCondominio:nroArea:tipo{urbano|area verde|parque|rural}
EJEMPLO => MODMAN:bolita@condominio.com:123:Laguna Azul:1:area verde
Listar todas las areas comunes de un condominio en particular
FORMATO => LISMAN:login:pass:nomCondominio
EJEMPLO => LISMAN:bolita@condominio.com:123:Laguna Azul
Modificar la cantidad de un insumo de una formula existente
FORMATO => DELMAN:loginAdmin:passAdmin:nomCondominio:nroArea
EJEMPLO => DELMAN:bolita@condominio.com:123:Laguna Azul:1
Añadir casa a un manzano en particular
FORMATO => ADDCASA:loginAdmin:passAdmin:nomCondominio:nroArea:nroCasa:dimension
EJEMPLO => ADDCASA:bolita@condominio.com:123:Laguna Azul:1:1:30x20mts
Modificar atributos de la casa //modifica solo la dimension
FORMATO => MODCASA:loginAdmin:passAdmin:nomCondominio:nroArea:nroCasa:dimension
EJEMPLO => MODCASA:bolita@condominio.com:123:Laguna Azul:1:1:30x30mts
Listar todas las casas de un manzano
FORMATO => LISCASA:login:pass:nomCondominio:nroArea
EJEMPLO => LISCASA:bolita@condominio.com:123:Laguna Azul:1
CU4: GESTIONAR COMUNICADOS
Añadir un comunicado
FORMATO => ADDCOMUN:loginComite:passComite:fecha:hora:asunto:tipo{reunion|actividades|recomendacion}
EJEMPLO => ADDCOMUN:bolita@condominio.com:123:2022-01-10:1730:Instalacion de camaras de seguridad:reunion
Listar todos los comunicados
FORMATO => LISCOMUN:login:pass:tipo{reunion|actividad|recomendacion|*}
EJEMPLO => LISCOMUN:bolita@condominio.com:123:*
Eliminar un comunicado
FORMATO => DELCOMUN:loginComite:passComite:idComunicado
EJEMPLO => DELCOMUN:bolita@condominio.com:123:1
Añadir un acta para un comunicado //solo se añade si el tipo de comunicado que pertenece a una reunion
FORMATO => ADDACTA:loginComite:passComite:idComunicado:tipo{descriptiva|informativa}:accion:responsable:[conclusion]
EJEMPLO => ADDACTA:bolita@condominio.com:123:1:descriptiva:Instalador de camaras de seguridad:Juan Perez
Listar todas las actas de un comunicado
FORMATO => LISACTA:login:pass:idComunicado
EJEMPLO => LISACTA:bolita@condominio.com:123:1
Listar todos los usuarios que asistieron a dicho comunicado de tipo reunion
FORMATO => DETACTA:loginComite:passComite:idComunicado
EJEMPLO => DETACTA:bolita@condominio.com:123:1
Añadir un comentario en la reunion de un comunicado de tipo reunion
FORMATO => ADDCOMEN:login:pass:idComunicado:[comentario]
EJEMPLO => ADDCOMEN:bolita@condominio.com:123:1
CU5: GESTIONAR MENSAJES
Registrar un mensaje
FORMATO => REGMEN:login:pass:asunto{reclamo|sugerencia|personal}:body:fecha:estado{curso|enviado}
EJEMPLO => REGMEN:ericka@condominio.com:ericka123:reclamo:estoy en desacuerdo con ese comunicado:2022-01-25:curso
Modificar atributos de un mensaje //solo modificas si el registro estuvo en curso
FORMATO => MODMEN:login:pass:idMensaje:asunto{reclamo|sugerencia|personal}:body:fecha:estado{curso|enviado}
EJEMPLO => MODMEN:ericka@condominio.com:ericka123:1:reclamo:ahora si estoy de acuerdo:2022-02-25:enviado
Eliminar un mensaje en particular //No importa si ya se envio lo elimina
FORMATO => DELMEN:login:pass:idMensaje
EJEMPLO => DELMEN:ericka@condominio.com:ericka123:1
Listar todos los mensajes //metodo para el usuario de tipo comite
FORMATO => LISMEN:loginComite:passComite:asunto{reclamo|sugerencia|personal|*}
EJEMPLO => LISMEN:bolita@condominio.com:123:reclamo
Listar todos los mensajes que escribio el usuario //mostraria de cualquier estado
FORMATO => LISUSERMEN:login:pass:estado{curso|enviado|*}
EJEMPLO => LISUSERMEN:ericka@condominio.com:ericka123:*
CU6: GESTIONAR SERVICIOS
Registrar un servicio publico
FORMATO => REGSERVICE:loginAdmin:PassAdmin:nom:monto:fechaIni:fechaFin
EJEMPLO => REGSERVICE:bolita@condominio.com:123:servicio de guardia:4000:2022-01-02:2022-12-02
Modificar un servicio publico
FORMATO => MODSERVICE:loginAdmin:PassAdmin:idServicio:nom:monto:fechaIni:fechaFin
EJEMPLO => MODSERVICE:simon@condominio.com:simon123:1:servicio de guardia:5500:2022-12-02:2023-12-02
Listar el historial de todos los servicios publicos añadidos
FORMATO => LISSERVICE:loginAdmin:PassAdmin
EJEMPLO => LISSERVICE:simon@condominio.com:simon123
Comando para cobrar el servicio a los dueños
FORMATO => ALTASERVICE:loginAdmin:PassAdmin:idServicio
EJEMPLO => ALTASERVICE:simon@condominio.com:simon123:1
Comando para No cobrar el servicio a los dueños
FORMATO => BAJASERVICE:loginAdmin:PassAdmin:idServicio
EJEMPLO => BAJASERVICE:simon@condominio.com:simon123:1
CU7: GESTIONAR PAGOS
Listar los servicios a pagar del usuario
FORMATO => LISPAGO:login:pass
EJEMPLO => LISPAGO:soto@condominio.com:soto123
Listar el historial de servicios pagados
FORMATO => HISPAGO:login:pass
EJEMPLO => HISPAGO:soto@condominio.com:soto123
Pagar uno o varios servicios
FORMATO => REGPAGO:login:pass:idServicio1:idServicio2:idServicio3:...
EJEMPLO => REGPAGO:soto@condominio.com:soto123:3:7
CU8: GESTIONAR REPORTES Y ESTADISTICAS
Listar todas las areas comunes que no estan registradas en un condominio en particular
FORMATO => LISNOMAN:login:pass
EJEMPLO => LISNOMAN:bolita@condominio.com:123
Listar todos los dueños propietarios de un manzano en particular
FORMATO => LISMANPROP:login:pass:nomCondominio:nroArea
EJEMPLO => LISMANPROP:bolita@condominio.com:123:Laguna Azul:1
Listar todos servicios no pagados en mora de todos los clientes
FORMATO => LISMORASERVICE:loginAdmin:passAdmin:nomCondominio:nroArea
EJEMPLO => LISMORASERVICE:bolita@condominio.com:123:Laguna Azul:1
